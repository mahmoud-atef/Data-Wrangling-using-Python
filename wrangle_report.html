Data Wrangling Report


WeRateDogs is a Twitter account that rates people's dogs with a humorous comment about the dog. These ratings almost always have a denominator of 10.This dataset contains basic tweet data (tweet ID, timestamp, text, etc.) for all 5000+ of their tweets as they stood on August 1, 2017.


Data wrangling, which consists of:

  • Gathering data

  • Assessing data

  • Cleaning data

Gathering :

The data consists of three sources:

1.Enhanced Twitter Archive

which contains basic tweet data for 2356 of their tweets. The WeRateDogs Twitter archive downloaded directly.

2.Image Predictions File

The tweet image predictions which is a table full of image predictions what breed of dog. downloaded programmatically using requests library.

3.Additional Data via the Twitter API

Back to the basic-ness of Twitter archives: retweet count and favorite count are two of the notable column omissions so Query the Twitter API for each tweet's JSON data using Python's tweepy library. specifically the tweet IDs within it
